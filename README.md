# mibu

**mibu** is pure Zig library for low-level terminal manipulation.

## Features
- Allocation free.
- Raw mode.
- 8-16 colors.
- True Color (24-bit RGB).
- Cursor controls.
- Clear(Erase) functions.
- Colors.
- Key events.
